🛡️ WSL - GitHub SSH 직접 연결 가이드 (전문가용)
이 가이드는 WSL(Linux) 환경에서 비밀번호 입력 없이 안전하게 GitHub와 통신하기 위한 SSH(Secure Shell) 설정 방법을 다룹니다.

1. SSH 키 생성 (Key Generation)
먼저 WSL 터미널을 열고 본인의 GitHub 계정 이메일을 사용하여 고유한 암호화 키를 생성합니다.

Bash
# -t: 암호화 알고리즘 (ed25519 권장), -C: 주석(이메일)
ssh-keygen -t ed25519 -C "your-email@example.com"
Enter file in which to save the key: 그냥 Enter (기본 경로 사용)

Enter passphrase: 보안을 위해 비워두려면 Enter를 두 번 입력합니다.

2. 생성된 공개키(Public Key) 확인 및 복사
생성된 두 개의 키 중 .pub 확장자가 붙은 공개키의 내용을 확인합니다.

Bash
cat ~/.ssh/id_ed25519.pub
주의: ssh-ed25519 AAA...로 시작하는 한 줄 전체를 정확히 복사하세요.

3. GitHub 계정에 SSH 키 등록
이제 복사한 키를 GitHub 서버에 알려주어야 합니다.

GitHub 접속: Settings > SSH and GPG keys

새 키 추가: New SSH Key 버튼 클릭

내용 입력:

Title: 구분하기 쉬운 이름 (예: WSL-Ubuntu-Laptop)

Key type: Authentication Key (기본값)

Key: 복사한 공개키 내용을 붙여넣기

저장: Add SSH Key 클릭

4. 연결 상태 최종 확인
설정이 완료되었다면 아래 명령어로 GitHub 서버와 정상적으로 통신하는지 테스트합니다.

Bash
ssh -T git@github.com
Hi [YourUsername]! You've successfully authenticated... 라는 메시지가 나오면 성공입니다!

## dev container docker
만약 TERMINAL 패널이 보이지 않으시면 Ctrl + Shift + `을 눌러주세요.
먼저 아래 명령어로 기존에 설치된 Docker 를 제거합니다.


sudo apt-get remove docker docker-engine docker.io containerd runc
snap remove docker



의존성 패키지를 설치합니다


sudo apt-get update
sudo apt-get install \
    ca-certificates \
    curl \
    gnupg \
    lsb-release

Docker 를 설치합니다

sudo mkdir -m 0755 -p /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
 
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt-get update

sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin


Docker 설치 결과를 확인합니다

docker --version
Docker 명령어 permission denied 에러가 발생할 경우 아래 명령어를 수행해주세요.

sudo chmod 777 /var/run/docker.sock
sudo usermod -aG docker $USER
newgrp docker

(VS Code에서) WSL ssh-key 발급
ssh-keygen -t rsa
eval $(ssh-agent)
ssh-add ~/.ssh/id_rsa
